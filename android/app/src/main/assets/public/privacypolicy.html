Política de Privacidad

FuelSmart utiliza Health Connect únicamente para leer y mostrar datos de salud que el usuario autoriza explícitamente en el dispositivo.

Datos que podemos leer

  • Pasos diarios
  • Calorías quemadas
  • Peso corporal

Uso de los datos

Los datos se usan para mejorar la experiencia dentro de la app, mostrar progreso y permitir que el usuario importe su peso actual con confirmación.

Almacenamiento

FuelSmart no vende datos personales a terceros. Los datos de salud se manejan conforme a la autorización del usuario y a las capacidades locales de Health Connect.
